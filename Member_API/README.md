## 1. API & DB 설계

🚩기능1 : **Member API**
  1. 로그인 
  2. 회원 가입
  3. 회원 탈퇴 
  4. 회원 정보 조회 (개별 조회)
<br>

🚩DB1 : **Member API**
  1. id
  2. email
  3. name
  4. password
  5. contact
<br>

🚩기능2 : **Channel API**
  1. 채널 생성 
  2. 채널 삭제 
  3. 채널 수정
  4. 채널 전체 조회
  5. 채널 개별 조회 
<br>

🚩DB2 : **Channel API**
  1. id
  2. name
  3. sub_name
  4. video_count
  5. user_id
<br>
